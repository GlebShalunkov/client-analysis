# 📊 Клиентский анализ (оконные функции SQL)

**Проект:** Анализ клиентской базы IT-фрилансера с использованием оконных функций SQL  
**Автор:** GlebShalunkov  
**Технологии:** Python, Pandas, SQL (оконные функции), Matplotlib

## 📌 О проекте
Анализ данных клиентов: ранжирование, поиск повторных заказов, сегментация.

## 🎯 Что сделано
- Ранжирование клиентов по сумме заказов (RANK, DENSE_RANK)
- Поиск повторных заказов в течение 30 дней (LAG/LEAD)
- Топ-3 клиента по каждому типу проектов (PARTITION BY)
- Визуализация результатов

## 🔥 Ключевые инсайты
- Топ-20% клиентов приносят 60% выручки
- 40% клиентов возвращаются в течение 30 дней
- Самые популярные типы проектов: дашборды и парсинг

## 📁 Файлы
- `client_analysis.ipynb` — основной ноутбук
- `client_projects.csv` — исходные данные
- `client_analysis_report.md` — подробный отчёт
